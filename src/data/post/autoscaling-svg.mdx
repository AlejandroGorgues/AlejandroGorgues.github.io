In case you have an SVG that want to get draw in your page without the need to resize it manually for every screen size by css, js or html, there exists an easy solution, 
by using `preserveAspectRatio="xMidYMid meet"` property instead of the width/height properties.

For example, we can replace:
```html
<svg width="12" height="7" viewBox="0 0 12 7" fill="none" xmlns="http://www.w3.org/2000/svg">
    <path fill-rule="evenodd" clip-rule="evenodd" d="{...}" fill="#3B3EC9"/>
</svg>
```
with:
```html
<svg preserveAspectRatio="xMidYMid meet" viewBox="0 0 12 7" fill="none" xmlns="http://www.w3.org/2000/svg">
        <path fill-rule="evenodd" clip-rule="evenodd" d="{...}" fill="#3B3EC9"/>
</svg>
```
> Keep in mind that we still need to set the height/width in a css file.
